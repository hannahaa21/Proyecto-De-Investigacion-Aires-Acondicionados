Agregar un aire acondicionado

Esta opción te permite agregar un nuevo aire acondicionado a partir de un nombre, Red de conexion y temperatura inicial dada


Configura un aire acondicionado

Esta opción te permite modificar un aire acondicionado especifico, en el cual se puede configurar de forma automática o manual, en el modo automático se fija un tiempo u hora de llegada, y a partir de la temperatura fija configurada el sistema configura tu aire con un ahorro de energia a partir del tiempo deseado


Revisa tus aires

Esta opción te permite revisar toda la información de tus aires, el nombre de este, la conexión y la configuración de temperatura impuesta


Información del usuario registrado

Esta opción te permite revisar toda la información de tu registro, tus datos personales y tu información principal
